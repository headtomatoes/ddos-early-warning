# Real-time L3/L4 DDoS Early Warning System using Lightweight Machine Learning

## Architecture
* `/xdp` - eBPF/XDP programs for packet inspection and fast dropping.
* `/ml` - Lightweight machine learning models for anomaly detection.
* `/controller` - Userspace application linking the eBPF maps to the ML models.
* `/grafana` - Dashboards and configurations for real-time monitoring.
